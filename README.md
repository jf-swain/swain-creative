# Swain-creative

Personal website

## Installing

Install project

```
  yarn
```

Run Project

```
  yarn run dev
```

## Build With

* [SASS](https://sass-lang.com/) - The css preprocessor used
* [Jest](https://jestjs.io/) - The js test runner
* [Next.js](https://github.com/zeit/next.js/) - Framework Seo react
